# 个人资产管理 App 使用说明

## 功能概述

本应用是一款简洁高效的个人资产管理工具，支持多币种资产录入、汇率自动转换、数据导入导出、跨设备同步和个税计算等功能。

## 主要功能

### 1. 资产管理

#### 新增资产
- 点击顶部导航栏的 **+** 按钮
- 填写资产信息：
  - **名称**：资产名称（如"存款"、"股票"等）
  - **金额**：支持数字或四则运算表达式（如"10000"、"5000+3000"）
  - **货币**：人民币(CNY)、美元(USD)、港币(HKD)
  - **类型**：存款、现金、股票、基金、房产、其他
- 点击确认保存

#### 编辑资产
- 在资产列表中点击任意资产项
- 进入详情页面后点击右上角 **编辑** 按钮
- 修改相关信息后确认保存

#### 删除资产
- 点击资产进入详情页面
- 点击右上角 **删除** 按钮
- 确认删除操作

**重要说明：**
删除资产时会自动清理该资产的所有历史操作记录，避免后续创建新资产时ID重复导致历史记录混淆。删除操作本身不会创建历史记录。

#### 查看资产详情
- 点击资产列表中的任意项目
- 查看资产的详细信息、历史操作记录和汇率转换后的价值

### 2. 资产分类显示

资产按类型分组显示，每组显示：
- 资产类型名称
- 该类别的总资产金额（已转换为人民币）
- 该类别下的所有资产项

支持的类型：
- 存款
- 现金
- 股票
- 基金
- 房产
- 其他

### 3. 汇率功能

#### 查看汇率
- 点击顶部导航栏的 **刷新** 按钮
- 弹出对话框显示当前美元和港币兑人民币汇率
- 显示上次更新时间

#### 刷新汇率
- 在汇率对话框中点击 **刷新汇率** 按钮
- 应用会从网络获取最新汇率
- 系统每小时自动更新一次汇率
- 如网络失败，会使用缓存的汇率或默认值

### 4. 导入导出功能

#### 导出资产
- 点击顶部导航栏的 **Share** 图标
- 选择 **导出**
- 选择保存位置和文件名
- 资产数据将以 JSON 格式导出

#### 导入资产
- 点击顶部导航栏的 **Share** 图标
- 选择 **导入**
- 选择之前导出的 JSON 文件
- 导入成功后会替换当前所有资产数据

#### 使用说明
- 点击 **导入/导出** 对话框中的 **使用说明** 按钮
- 查看详细的文件同步功能说明
- 了解使用步骤、注意事项和文件格式说明

### 5. 二维码同步功能

#### 生成二维码（发送端）
1. 点击顶部导航栏的 **二维码** 图标
2. 选择 **生成**
3. 应用启动本地 HTTP 服务器
4. 显示包含同步信息的二维码
5. 使用另一台设备扫描此二维码即可接收资产数据

#### 扫描二维码（接收端）
1. 点击顶部导航栏的 **二维码** 图标
2. 选择 **扫描**
3. 授权相机权限
4. 扫描发送端设备上显示的二维码
5. 自动下载并导入资产数据

#### 使用说明
- 点击 **二维码同步** 对话框中的 **使用说明** 按钮
- 查看详细的二维码同步功能说明
- 了解使用步骤、安全注意事项和技术原理

**同步原理：**
- 发送端在本地网络启动 HTTP 服务器
- 二维码包含服务器地址、会话ID和加密密钥
- 数据经过 AES 加密传输，确保安全
- 两台设备需在同一局域网内

### 6. 资产趋势图表

#### 查看资产走势
- 在主页总资产卡片右侧点击 **趋势** 按钮
- 可选择按天、周、月、年查看资产变化曲线
- 横轴为时间，纵轴为总资产金额（人民币）

### 7. 税率计算器

应用提供税率计算器，帮助您计算不同类型的个人所得税：

- **年终奖税率计算器**：计算年终奖个人所得税，支持当月工资并入计税
- **普通收入税率计算器**：计算月薪或年薪的综合所得税，考虑社保公积金和专项附加扣除

#### 年终奖税率计算器

##### 访问计算器
- 点击底部导航栏的 **税率计算** 选项
- 选择 **年终奖计算器** 选项卡

##### 计算税费
1. 输入年终奖金额
2. （可选）输入当月工资，可选择是否将当月工资并入计税
3. 点击 **计算税费** 按钮
4. 查看计算结果，包括：
   - 年终奖金额
   - 当月工资（如输入）
   - 合计收入
   - 适用税率
   - 应纳税额
   - 速算扣除数
   - 应缴个税
   - 最终税率（交的税/总金额）
   - 税后收入

##### 计税规则
- 采用单独计税方式（年终奖除以12个月确定税率）
- 当月工资低于5000元时，可从年终奖中扣除差额
- 税率表（根据年终奖除以12后的金额确定）：
  - 不超过3,000元：3%
  - 超过3,000元至12,000元：10%
  - 超过12,000元至25,000元：20%
  - 超过25,000元至35,000元：25%
  - 超过35,000元至55,000元：30%
  - 超过55,000元至80,000元：35%
  - 超过80,000元：45%

##### 使用示例
- 年终奖36,000元，当月工资8,000元：
  - 年终奖除以12：3,000元
  - 适用税率：3%
  - 应缴个税：36,000 × 3% = 1,080元
  - 税后收入：36,000 + 8,000 - 1,080 = 42,920元

#### 普通收入税率计算器

##### 访问计算器
- 点击底部导航栏的 **税率计算** 选项
- 选择 **普通收入计算器** 选项卡

##### 计算税费
1. 选择计算模式：月薪或年薪
2. 输入工资金额
3. 设置社保公积金比例（默认值已预设）：
   - 养老保险个人比例：8%
   - 公积金个人比例：12%
   - 医疗保险个人比例：2%
   - 失业保险个人比例：0.5%
4. 输入专项附加扣除金额
5. 点击 **计算税费** 按钮
6. 查看计算结果，包括：
   - 月薪/年薪
   - 社保公积金扣除
   - 专项附加扣除
   - 应纳税所得额
   - 适用税率
   - 速算扣除数
   - 应缴个税
   - 税后收入

##### 计税规则
- 采用综合所得税率表
- 起征点：5,000元/月
- 社保公积金在税前扣除
- 专项附加扣除包括子女教育、继续教育、大病医疗、住房贷款利息、住房租金、赡养老人等

##### 税率表（年度应纳税所得额）：
  - 不超过36,000元：3%
  - 超过36,000元至144,000元：10%
  - 超过144,000元至300,000元：20%
  - 超过300,000元至420,000元：25%
  - 超过420,000元至660,000元：30%
  - 超过660,000元至960,000元：35%
  - 超过960,000元：45%


### 8. 表达式计算功能

资产金额支持四则运算表达式，便于动态计算：
- 简单数字：`10000`
- 加法：`5000+3000`
- 减法：`10000-2000`
- 乘法：`100*5`
- 除法：`10000/2`
- 复合表达式：`5000+3000*2-1000`

示例场景：
- 分散存款：`5000+3000+2000`（三个账户总和）
- 股票估值：`100*12.5`（100股，每股12.5元）

### 9. 清空所有数据

**警告：此操作不可恢复！**

- 点击顶部导航栏的 **删除** 图标
- 确认后清空：
  - 所有资产数据
  - 资产操作记录
  - 总资产历史快照

## 技术说明

### 数据存储
- 使用 Android SharedPreferences 本地存储
- 支持资产列表、操作记录、汇率缓存的持久化
- 资产历史记录管理：
  - 创建、更新资产时自动记录操作历史
  - 删除资产时自动清理相关历史记录，避免ID重复混淆
  - 历史记录按资产ID关联存储

### 网络功能
- 汇率数据来源：https://www.huilvbiao.com
- 支持美元(USD)和港币(HKD)兑人民币(CNY)汇率
- 使用 OkHttp 进行网络请求

### 安全性
- 资产同步使用 AES-128 加密
- 二维码数据包含一次性会话ID
- 局域网内通信，支持 HTTPS 和 HTTP

### 系统要求
- Android 7.0 (API 24) 及以上
- 相机权限（用于扫描二维码）
- 网络权限（用于获取汇率和同步）

## 常见问题

**Q: 为什么汇率刷新失败？**
A: 检查网络连接，应用会使用缓存的汇率。如缓存过期，会使用默认汇率（USD=7.2, HKD=0.92）。

**Q: 二维码同步无法连接？**
A: 确保两台设备在同一局域网，并且发送端设备已开启局域网HTTP权限。

**Q: 导入数据失败？**
A: 确保导入的文件是本应用导出的标准JSON格式。

**Q: 支持云同步吗？**
A: 目前仅支持局域网WiFi直连同步，未来版本可能增加云同步功能。

**Q: 删除资产后，为什么看不到该资产的过往操作记录？**
A: 这是设计行为。删除资产时会自动清理该资产的所有历史操作记录，避免后续创建新资产时ID重复导致历史记录混淆。如需保留历史记录，建议在删除前导出资产数据备份。

**Q: 税率计算器的计算结果准确吗？**
A: 税率计算器基于中国个人所得税法规定设计，计算结果仅供参考。实际纳税金额可能因地方政策、特殊扣除等因素有所不同，建议以税务机关的最终核定为准。

**Q: 普通收入税率计算器中的社保公积金比例可以调整吗？**
A: 可以。计算器提供了默认比例（养老保险8%、公积金12%、医疗保险2%、失业保险0.5%），您可以根据实际情况调整这些比例。
